## 🎮 &nbsp;About

**Mysterious City V2** là source backup của thành phố RolePlay trong **GTA V FiveM**, được xây dựng trên nền tảng **QBCore Framework**. Chúng tôi rất hoan nghênh các bạn đến tham gia cộng đồng hoặc gia nhập Team để cùng phát triển!

> *"Every player has a story. What's yours?"*

<br/>

## 🏗️ &nbsp;Server Architecture

```
MysteriousCity-V2/
├── 🎯 [qb]              # QBCore Framework & core resources
├── 🗺️ [map]             # Custom maps & MLOs
├── 🏘️ [defaultmaps]     # Default map modifications
├── 🚗 [veh]             # Custom vehicles
├── 👤 [peds]            # Custom character models
├── 👕 [eup]             # Emergency Uniforms Pack
├── 🔫 [weaponskin]      # Custom weapon skins
├── 🎙️ [voice]           # Voice chat system
├── 📊 [hud]             # Custom HUD & UI
├── 🔧 [standalone]      # Standalone scripts
├── ⚙️ [cfx-default]     # CitizenFX default configs
├── 📡 [Ping]            # Ping system
├── 🤖 zdiscord          # Discord integration bot
├── 🏥 qb-diease         # Disease system
└── 📦 [unused]          # Archived/unused resources
```

## 🛠️ &nbsp;Installation

```bash
# 1. Clone the repository
git clone https://github.com/ngoclonglam/MysteriousCity-V2.git

# 2. Copy resources to your FiveM server's resources folder
cp -r MysteriousCity-V2/* /path/to/your/fivem-server/resources/

# 3. Import SQL files to your database (if any)

# 4. Add resources to server.cfg
# ensure [qb]
# ensure [map]
# ensure [hud]
# ... etc

# 5. Start your server
```
